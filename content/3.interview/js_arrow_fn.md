# \[ JavaScript ] 什麼是箭頭函式? 跟一般函式有什麼差別?
ES6 後出現的一種函式表現方式

箭頭函式和一般函式的主要差異有四點，我們將在下方詳細說明：

箭頭函式語法不同、寫法也較簡潔
箭頭函式沒有自己的 this、也無法直接修改 this 的指向
箭頭函式沒有自己的 arguments
箭頭函式不能作為構造函式使用
